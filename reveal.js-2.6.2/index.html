Overview of Cordova

  • Open source mobile development framework (free to use)
  • App is implemented as a web page ( index.html, style with css, add cool stuff Javascript)
  • Allow to add variety of plugins to access camera, battery, network.etc..

Here how to install Cordova CLI

on OS X and Linux

$ sudo npm install -g cordova

on Windows

$ npm install -g cordova

make sure you add npm to PATH.

Create the APP

Yup, we're done with the installation. I promise!!

First, go to directory you want to create your app folder: 

$ cordova create hello com.example.hello HelloWorld

  • com.example.hello is reverse domain-style indentifier (google).

    • Remember? Cordova works with multiple platforms!

    Here how to add platforms:

    $ cd hello

    Platforms you can add:

    $ cordova platform add ios
    $ cordova platform add amazon-fireos
    $ cordova platform add android
    $ cordova platform add blackberry10
    $ cordova platform add firefoxos

    So you've come this far! Congrats

    You can build your app and test it on emulator or physical device: 

    $ cordova build
    $ cordova build ios

    TEST:

    $ cordova emulate android
    $ cordova run android

    Plugins

    A plugin is a bit of add-on code that provides an interface to native components.

  • $ cordova plugin add org.apache.cordova.device
  • $ cordova plugin add org.apache.cordova.contacts
  • $ cordova plugin add org.apache.cordova.geolocation
  • $ cordova plugin add org.apache.cordova.camera
  • $ cordova plugin add org.apache.cordova.vibration

    • You can checkout more plugins in Cordova documentation.
