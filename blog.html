Show folders and file color in Cygwin (Window)

Posted on September 23, 2014

I have been trying to learn Vim and shell script. I found a cool trick to show folders and files color in Cygwin.
First, change current directory to Home directly by cd $HOME then open .bachrc file vim .bashrc,

Add the following line of code to .bashrc

alias ls='ls -A --color=auto --group-directories-first'
alias ll='ls -lh'
LS_COLORS='no=00:di=33;01:tw=33;01:ow=33;01'
LS_COLORS=$LS_COLORS':fi=00:ln=00:pi=00:so=00:bd=00:cd=00:or=00:mi=00:ex=00'
LS_COLORS=$LS_COLORS':*.sh=31:*.sh=31:*.exe=31:*.bat=31:*.com=31'
export LS_COLORS

Save and Close vim with ESC + :wq, then run this command on Cygwin ~ /.bash_profile
